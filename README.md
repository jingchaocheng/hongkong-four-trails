# 香港四径 - Hong Kong Four Trails

一个展示和规划香港四径徒步路线的网站。

## 功能特性

- 🏔️ 展示香港四径的详细介绍
- 🗺️ 交互式地图显示路径
- 📍 标记点导航（如 M001, M050 等）
- 📅 行程规划功能
- 📊 爬升高度图表
- 🎨 现代化响应式设计

## 技术栈

- React 18
- TypeScript
- Vite
- React Router
- Leaflet (地图)
- Tailwind CSS

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 使用说明

1. 首页展示四条径的基本信息
2. 点击任意径进入详情页
3. 在详情页可以：
   - 查看完整路径地图
   - 规划行程（选择日期和标记点）
   - 查看选定路径的轨迹和爬升高度

## 数据信息
- **露营点（42）**：渔农自然护理署指定营地 41 处、康乐及文化事务署营地 1 处
- **厕所（165，地图约 124）**：郊野公园厕所；其中 41 个为营地配套公厕，不单独标注
- **加水站（37）**：郊野公园加水站（全览/聚焦当天均按径道 0.6 公里内筛选）

## 士多 / 补水 / 厕所标注

1. 详情页左下角勾选「开启地图标注」
2. 点击地图放置，在弹窗里**多选设施**（士多 / 补水 / 厕所），同一地点勾多项即可，不必重复打点
3. 保存为本地草稿后，点「导出 JSON」或「下载文件」，覆盖到 `src/data/supplyPoints.json` 再提交

图层开关仅保留营地等其它选项；补给点默认全部显示。

## 底图

默认使用 **OpenStreetMap**。若瓦片加载失败，会自动切换到 **ArcGIS 街道**（也可在地图左下角手动切换）。
